# AI Ambulance Route Optimizer

A command-line Python route optimizer for ambulance navigation between Marikina-area medical locations. It uses Dijkstra's algorithm to find the shortest available route while showing risk information for each trip.

## Run

```bash
python main.py
```

## Features

- Lists available locations.
- Finds the best route between two locations.
- Displays total distance and risk score.
- Labels route risk as low, medium, or high.
- Lets the user reject a route and search for an alternate route.
